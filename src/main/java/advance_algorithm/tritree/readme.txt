前缀树（字典树）：
1. 一般用来存储字符串（匹配效率很高）
2. 子节点代表的字符串是由结点本身的原始字符串，以及通往该子节点路径上所有的字符组成的——根结点表示空字符
3.结点所有的后代都与该节点相关的字符串有着共同的前缀。
